üîç Globbing (Wildcards)

Glob characters are shell features that allow you to match filenames using patterns.

Basic Glob Characters

Character Matches Example Description
* Zero or more characters D* All files starting with D
? Exactly one character ?????? Files with exactly 6 characters
[abc] Any one of the characters [DP]* Files starting with D or P
[!abc] Any character except these [!DP]* Files NOT starting with D or P
[a-z] Range of characters [D-P]* Files starting with D through P

Examples:

echo *

Shows all files in current directory

echo D*

Shows files starting with D

echo *s

Shows files ending with s

echo D*n*s

Shows files starting with D, containing n, and ending with s

üìÅ File Operations

Copying Files

Command Description Example
cp source dest Basic copy cp file1.txt file2.txt
cp -p source dest Preserve attributes cp -p /etc/hosts hosts
cp -v source dest Verbose mode cp -v file1.txt file2.txt
cp -R source dest Recursive (directories) cp -R /etc/udev Myetc

Moving and Renaming

mv source destination

The mv command both moves and renames files.

Examples:

mv oldname newname

Rename a file

mv file.txt /home/user/

Move file to another directory

Deleting Files and Directories

Command Description Use Case
rm filename Delete file Remove single file
rm -r directory Delete directory recursively Remove directory and contents
rmdir directory Delete empty directory Remove empty directory only
Important: The period (.) represents the current directory, and the tilde (~) represents the home directory.

üìù Text Processing

Viewing Files

Command Description Usage
cat filename Display entire file Small files
more filename Page through file Large files (forward only)
less filename Page through file Large files (forward/backward)
head filename First 10 lines Beginning of file
tail filename Last 10 lines End of file

Examples:

head -5 /etc/passwd

Show first 5 lines

tail -3 /var/log/messages

Show last 3 lines

head -n -20 /etc/passwd

Show all but last 20 lines

Navigation in less/more

Key Action

b Previous page
/pattern Search forward
n Next match
N Previous match
q Quit

üîÑ I/O Redirection

Standard Streams

  • stdin (0): Standard input (keyboard)
  • stdout (1): Standard output (screen)
  • stderr (2): Standard error (screen)

Redirection Operators

Operator Description Example
> Redirect output (overwrite) echo "Hello" > file.txt
>> Redirect output (append) echo "World" >> file.txt
< Redirect input command < input.txt
| Pipe (send output to next command) ls | grep txt

Examples:

echo "Hello World" > message.txt

Create file with content

echo "Second line" >> message.txt

Append to file

ls /etc | tail -5

Show last 5 files in /etc

Warning: Using > overwrites existing files (called "clobbering"). Use >> to append instead.

üîç Grep & Search

Basic Grep Usage

grep pattern filename

Search for pattern in file and display matching lines.

Regular Expression Characters

Character Meaning Example Matches
^ Beginning of line ^root Lines starting with "root"
$ End of line bash$ Lines ending with "bash"
. Any single character r..t "root", "rout", "reat", etc.
* Zero or more of previous ro*t "rt", "rot", "root", "rooot"

Examples:

grep root /etc/passwd

Find all lines containing "root"

grep '^root' /etc/passwd

Find lines starting with "root"

grep 'bash$' /etc/passwd

Find lines ending with "bash"

cat /etc/passwd | grep sshd

Pipe output to grep

Tip: Use single quotes around patterns to prevent shell interpretation. Grep is "greedy" - it matches all instances of the pattern.

üñ•Ô∏è Shell Scripting

Basic Script Structure

#!/bin/bash # This is a comment echo "Hello, World!"

Key Elements

  • Shebang: #!/bin/bash (first line)
  • Comments: Lines starting with #
  • Commands: Regular Linux commands

Variables

name="John" # Assign variable echo $name # Use variable echo ${name} # Alternative syntax

User Input

read -p "Enter your name: " username echo "Hello, $username!"

Making Scripts Executable

chmod +x script.sh # Make executable ./script.sh # Run script

Complete Example Script:

#!/bin/bash # Simple greeting script read -p "What is your name? " name echo "Hello, $name!" echo "Welcome to Linux!"

Aliases

Create shortcuts for commands:

alias ll='ls -la' alias ..='cd ..' alias quit='exit'
Tip: Add aliases to ~/.bashrc file to make them permanent.

üîé Find Command

Basic Syntax

find [path] [options] [expression]

Common Options

Option Description Example
-name Find by filename find /home -name "*.txt"
-type f Find files only find /etc -type f
-type d Find directories only find /var -type d
-size Find by size find /home -size +100M
-delete Delete found files find /tmp -name "*.tmp" -delete

Examples:

find /home/student -name "test"

Find files named "test"

find /home/student -name "test" -delete

Find and delete files named "test"

find . -name "*.log" -type f

Find all .log files in current directory

Important: Find requires you to specify the starting path. Use . for current directory, / for entire system.

üí° Exam Tips

  • Practice typing commands - muscle memory is important
  • Remember that Linux is case-sensitive
  • Use tab completion to avoid typos
  • Read error messages carefully - they often tell you what's wrong
  • Practice with the vi editor - it's commonly tested
  • Know the difference between > and >> for redirection
  • Understand file permissions and the chmod command
