üî• TOP 14 MOST IMPORTANT Questions/Commands

1. What are the two modes of Vi editor and how do you switch between them?
‚Ä¢ Command mode and Insert mode
‚Ä¢ Press i or Insert to enter Insert mode
‚Ä¢ Press Esc to return to Command mode
2. How do you save and exit Vi editor?
‚Ä¢ :wq! = Save and exit
‚Ä¢ :q! = Exit without saving
‚Ä¢ :w = Save only
‚Ä¢ ZZ = Save and exit (no colon needed)
3. What are the three main glob (wildcard) characters and what do they match?
‚Ä¢ * = Matches zero or more characters
‚Ä¢ ? = Matches exactly one character
‚Ä¢ [abc] = Matches one character from the list
‚Ä¢ [a-z] = Matches one character from range
‚Ä¢ [!abc] = Matches one character NOT in the list
4. What's the difference between > and >> redirection?
‚Ä¢ > = Overwrites (clobbers) existing file content
‚Ä¢ >> = Appends to existing file content
Example: echo "Hello" > file.txt vs echo "World" >> file.txt
5. How do you copy, move, and remove files/directories?
‚Ä¢ cp source destination = Copy file
‚Ä¢ cp -r source dest = Copy directory recursively
‚Ä¢ mv source destination = Move/rename file
‚Ä¢ rm filename = Remove file
‚Ä¢ rm -r directory = Remove directory recursively
‚Ä¢ rmdir directory = Remove empty directory only
6. How do you make a script executable and run it?
‚Ä¢ chmod +x script.sh = Make executable
‚Ä¢ ./script.sh = Run script in current directory
‚Ä¢ Script must start with #!/bin/bash (shebang)
7. What are the standard streams and their file descriptors?
‚Ä¢ stdin (0) = Standard input (keyboard)
‚Ä¢ stdout (1) = Standard output (screen)
‚Ä¢ stderr (2) = Standard error (screen)
‚Ä¢ command 2> error.txt = Redirect errors to file
‚Ä¢ command > output.txt 2>&1 = Redirect both stdout and stderr
8. How do you search for text using grep?
‚Ä¢ grep "pattern" filename = Search in file
‚Ä¢ grep "^root" /etc/passwd = Search lines starting with "root"
‚Ä¢ grep "sync$" /etc/passwd = Search lines ending with "sync"
‚Ä¢ cat file | grep "text" = Search using pipe
‚Ä¢ grep -E "pattern1|pattern2" = Extended regex (OR operator)
9. What are the essential Vi navigation and editing commands?
‚Ä¢ h,j,k,l = Navigate left, down, up, right
‚Ä¢ dd = Delete (cut) current line
‚Ä¢ yy = Copy (yank) current line
‚Ä¢ p = Paste
‚Ä¢ u = Undo
‚Ä¢ G = Go to end of file
‚Ä¢ 1G = Go to beginning of file
‚Ä¢ /searchterm = Search forward
‚Ä¢ n = Next search result
10. How do you use pipes (|) and what's their purpose?
‚Ä¢ | = Pipe output of one command to input of another
‚Ä¢ ls | more = View long listings page by page
‚Ä¢ cut -d: -f1 /etc/passwd | sort = Extract usernames and sort
‚Ä¢ command | head -5 = Show only first 5 lines
‚Ä¢ command | tail -5 = Show only last 5 lines
11. How do you use find command to search for files?
‚Ä¢ find /path -name "filename" = Find by name
‚Ä¢ find /home -name "*.txt" = Find all .txt files
‚Ä¢ find ~ -name "bash" = Find in home directory
‚Ä¢ find /etc -name hosts = Find specific file
Note: find searches recursively through subdirectories
12. What are the basic shell script conditional statements?
‚Ä¢ if [ condition ]; then ... else ... fi
‚Ä¢ [ $age -lt 16 ] = Numeric less than
‚Ä¢ [ $age -gt 18 ] = Numeric greater than
‚Ä¢ [ $age -eq 21 ] = Numeric equal to
‚Ä¢ test $age -lt 16 = Alternative syntax
Important: Spaces around brackets are required!
13. How do you view large files without loading them entirely?
‚Ä¢ more filename = View page by page (spacebar = next page)
‚Ä¢ less filename = Enhanced viewer (q to quit)
‚Ä¢ head filename = Show first 10 lines
‚Ä¢ head -5 filename = Show first 5 lines
‚Ä¢ tail filename = Show last 10 lines
‚Ä¢ tail -5 filename = Show last 5 lines
14. How do you create and use variables in shell scripts?
‚Ä¢ name="value" = Create variable (no spaces around =)
‚Ä¢ echo $name = Use variable with $ prefix
‚Ä¢ read name = Read user input into variable
‚Ä¢ read -p "Enter name: " name = Read with prompt
Example: echo "Hello, $name!"

‚ö° 12 LESS IMPORTANT (But Still Valuable) Questions

15. How do you create directories and empty files?
‚Ä¢ mkdir dirname = Create directory
‚Ä¢ mkdir -p path/to/dir = Create nested directories
‚Ä¢ touch filename = Create empty file
‚Ä¢ touch file1 file2 file3 = Create multiple files
16. What are some useful Vi text manipulation commands?
‚Ä¢ dw = Delete word
‚Ä¢ 2dw = Delete 2 words
‚Ä¢ x = Delete character under cursor
‚Ä¢ X = Delete character before cursor
‚Ä¢ D = Delete to end of line
‚Ä¢ J = Join current line with next line
‚Ä¢ yw = Yank (copy) word
‚Ä¢ P = Paste before cursor
17. How do you use loops in shell scripts?
‚Ä¢ For loop: for i in 1 2 3; do echo $i; done
‚Ä¢ While loop: while [ $num -le 100 ]; do ... done
‚Ä¢ Seq command: for i in $(seq 1 10); do touch test$i; done
‚Ä¢ File loop: for file in /etc/*; do echo $file; done
18. What are regular expression patterns in grep?
‚Ä¢ . = Matches any single character
‚Ä¢ ^ = Matches beginning of line
‚Ä¢ $ = Matches end of line
‚Ä¢ [0-9] = Matches any digit
‚Ä¢ [a-z] = Matches any lowercase letter
‚Ä¢ {3} = Repeat pattern 3 times (needs -E option)
19. How do you copy files with specific options?
‚Ä¢ cp -v source dest = Verbose mode (show what's being done)
‚Ä¢ cp -p source dest = Preserve timestamps and permissions
‚Ä¢ cp source . = Copy to current directory (. = current dir)
‚Ä¢ cp -R source dest = Copy directories recursively
20. What are different ways to insert text in Vi?
‚Ä¢ i = Insert before cursor
‚Ä¢ a = Insert after cursor
‚Ä¢ o = Open new line below current line
‚Ä¢ O = Open new line above current line
‚Ä¢ A = Insert at end of current line
‚Ä¢ I = Insert at beginning of current line
21. How do you search and replace in Vi?
‚Ä¢ :%s/old/new/g = Replace all occurrences in file
‚Ä¢ :s/old/new/ = Replace first occurrence in current line
‚Ä¢ :1,10s/old/new/g = Replace in lines 1-10
‚Ä¢ /searchterm = Search forward
‚Ä¢ n = Next occurrence, N = Previous occurrence
22. How do you handle errors in commands and scripts?
‚Ä¢ command 2>/dev/null = Ignore error messages
‚Ä¢ command 2>error.log = Save errors to file
‚Ä¢ if grep $name /etc/passwd > /dev/null; then ...
‚Ä¢ Check exit codes: commands return 0 for success, non-zero for failure
23. What are useful file listing and viewing commands?
‚Ä¢ ls -l = Long format listing
‚Ä¢ ls -la = Show hidden files too
‚Ä¢ cat filename = Display entire file
‚Ä¢ wc filename = Count lines, words, characters
‚Ä¢ wc -l filename = Count lines only
24. How do you extract fields from delimited files?
‚Ä¢ cut -d: -f1 /etc/passwd = Extract first field (username)
‚Ä¢ cut -d: -f1,3 /etc/passwd = Extract fields 1 and 3
‚Ä¢ cut -c1-5 filename = Extract characters 1-5
‚Ä¢ Often used with pipe: cat file | cut -d, -f2
25. How do you create and manage aliases?
‚Ä¢ alias shortname='long command' = Create alias
‚Ä¢ alias ll='ls -la' = Example alias
‚Ä¢ alias = Show all aliases
‚Ä¢ Add to ~/.bashrc for permanent aliases
‚Ä¢ Example: alias uit='shutdown -h now'
26. What are file permission commands you should know?
‚Ä¢ chmod +x file = Make file executable
‚Ä¢ chmod 755 file = Set specific permissions
‚Ä¢ chmod a+x file = Make executable for all users
‚Ä¢ ls -l = View file permissions
Scripts need execute permission to run with ./script.sh
