Linux System Administration Exam - Questions and Answers

Section 1: Globbing and Wildcards

Question 1

Write commands using globbing to accomplish the following tasks:
a) List all files in the current directory that start with the letter 'D'
Answer: ls D*
b) List all files that are exactly 6 characters long
Answer: ls ??????
c) List all files that start with either 'a' or 'b'
Answer: ls [ab]*

Question 2

Explain what each of the following glob patterns matches:
a) *s
Answer: Matches all files that end with the letter 's'.
b) [!DP]*
Answer: Matches all files that do NOT start with 'D' or 'P'.
c) D*n*s
Answer: Matches files that start with 'D', contain 'n' somewhere, and end with 's'.

Question 3

Write a command to display all files in the /etc directory that start with any letter from 'a' to 'm' and end with '.conf'
Answer: ls /etc/[a-m]*.conf

Section 2: File Operations

Question 4

Write the complete sequence of commands to:
a) Create a directory called 'backup' in your home directory
Answer: mkdir ~/backup
b) Copy the file /etc/hosts to the backup directory, preserving all attributes
Answer: cp -p /etc/hosts ~/backup/
c) Rename the copied file to 'hosts_backup'
Answer: mv ~/backup/hosts ~/backup/hosts_backup
d) List the contents of the backup directory with detailed information
Answer: ls -la ~/backup/

Question 5

Explain the difference between the following commands:
a) cp -p source destination
Answer: Preserves file attributes (timestamps, permissions).
b) cp -R source destination
Answer: Copies directories recursively (including subdirectories).
c) cp -v source destination
Answer: Shows verbose output (displays what is being copied).

Question 6

Write commands to:
a) Remove a file called 'tempfile.txt'
Answer: rm tempfile.txt
b) Remove a directory called 'olddir' and all its contents
Answer: rm -r olddir
c) Remove an empty directory called 'emptydir'
Answer: rmdir emptydir

Section 3: Text Processing and I/O Redirection

Question 7

Write commands to:
a) Display the first 5 lines of /etc/passwd
Answer: head -5 /etc/passwd
b) Display the last 3 lines of /var/log/messages
Answer: tail -3 /var/log/messages
c) Display lines 10-15 of a file called 'data.txt' (use head and tail with pipe)
Answer: head -15 data.txt | tail -6
d) Count the number of lines in /etc/passwd
Answer: wc -l /etc/passwd

Question 8

Explain the difference between:
a) echo "Hello World" > file.txt
Answer: Overwrites the file completely with new content.
b) echo "Hello World" >> file.txt
Answer: Appends content to the end of the file.
c) What happens if file.txt doesn't exist in each case?
Answer: If file.txt doesn't exist, both commands will create it.

Question 9

Write a command to:
a) List all files in /etc and save the output to a file called 'etc_files.txt'
Answer: ls /etc > etc_files.txt
b) List all files in /home and append the output to 'etc_files.txt'
Answer: ls /home >> etc_files.txt
c) Display the total number of lines in 'etc_files.txt'
Answer: wc -l etc_files.txt

Section 4: Grep and Search Commands

Question 10

Write grep commands to:
a) Find all lines in /etc/passwd that contain the word 'root'
Answer: grep root /etc/passwd
b) Find all lines in /etc/passwd that start with 'root'
Answer: grep '^root' /etc/passwd
c) Find all lines in /etc/passwd that end with 'bash'
Answer: grep 'bash$' /etc/passwd
d) Find all lines in /etc/passwd that contain 'root' and display line numbers
Answer: grep -n root /etc/passwd

Question 11

Write find commands to:
a) Find all files named 'test.txt' in the /home directory
Answer: find /home -name test.txt
b) Find all .log files in the current directory and subdirectories
Answer: find . -name "*.log"
c) Find all directories named 'backup' starting from the root directory
Answer: find / -type d -name backup

You are editing a file in vi. Write the exact keystrokes to:

b) Exit insert mode and return to command mode
Answer: Esc

Answer: 1G

Answer: /error

Section 6: Shell Scripting

Question 14

Write a complete bash script that:
1. Asks the user for their name
2. Asks the user for their age
3. Displays a personalized greeting with their name and age
4. Creates a directory with their name
5. Creates a file called 'info.txt' in that directory containing their name and age
Include the proper shebang line and make sure to show how to make the script executable.
Answer: 
#!/bin/bash
echo "Welcome to the user information script!"
read -p "Please enter your name: " name
read -p "Please enter your age: " age
echo "Hello, $name! You are $age years old."
mkdir "$name"
echo "Name: $name" > "$name/info.txt"
echo "Age: $age" >> "$name/info.txt"
echo "Directory '$name' created with your information!"
# To make executable: chmod +x script.sh

b) Create an alias called 'quit' that runs 'exit'
