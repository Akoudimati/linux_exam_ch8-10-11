üîç Globbing (Wildcards)

1. What does the asterisk (*) glob character match?

  • Exactly one character
  • Zero or more characters
  • Only alphabetic characters
  • Only numeric characters
Correct! The asterisk (*) matches zero or more characters in a filename. For example, "D*" matches all files starting with "D".

2. What does the question mark (?) glob character match?

  • Exactly one character
  • Zero or more characters
  • A range of characters
  • No characters
Correct! The question mark (?) matches exactly one character, no more and no less. For example, "??????" matches files with exactly 6 characters.

3. What does the pattern [DP]* match?

  • Files containing D or P anywhere
  • Files ending with D or P
  • Files starting with D or P
  • Files with exactly D or P
Correct! [DP]* matches files that start with either D or P, followed by zero or more characters.

4. What does the pattern [!DP]* match?

  • Files starting with D or P
  • Files NOT starting with D or P
  • Files containing D and P
  • Empty files
Correct! The exclamation mark (!) negates the character class, so [!DP]* matches files that do NOT start with D or P.

5. Which command displays all files ending with 's'?

  • echo s*
  • echo ?s
  • echo [s]*
  • echo *s
Correct! echo *s displays all files ending with 's'. The asterisk matches zero or more characters before the 's'.
üìÅ File Operations

6. Which command copies a file while preserving its attributes?

  • cp -p source destination
  • cp -v source destination
  • cp -r source destination
  • cp -a source destination
Correct! The -p option preserves file attributes like timestamps and permissions when copying.

7. Which command recursively copies directories?

  • cp -p source destination
  • cp -v source destination
  • cp -R source destination
  • cp -l source destination
Correct! The -R (recursive) option allows copying directories and their contents.

8. What does the mv command do?

  • Only moves files
  • Moves and renames files
  • Only copies files
  • Only deletes files
Correct! The mv command both moves files to different locations and renames them.

9. Which command removes directories and their contents?

  • rm -r directory
  • rm directory
  • rmdir directory
  • del directory
Correct! rm -r removes directories and their contents recursively. rmdir only works on empty directories.

10. What does the period (.) represent in Linux?

  • Home directory
  • Root directory
  • Current directory
  • Parent directory
Correct! The period (.) represents the current directory in Linux.
üìù Text Processing & Redirection

11. What does the > symbol do in Linux?

  • Appends output to a file
  • Redirects output to a file (overwrites)
  • Reads input from a file
  • Pipes output to another command
Correct! The > symbol redirects output to a file, overwriting existing content.

12. What does the >> symbol do?

  • Appends output to a file
  • Redirects output to a file (overwrites)
  • Reads input from a file
  • Pipes output to another command
Correct! The >> symbol appends output to a file without overwriting existing content.

13. What does the | (pipe) symbol do?

  • Redirects output to a file
  • Appends output to a file
  • Reads input from a file
  • Sends output of one command to another
Correct! The pipe (|) sends the output of one command as input to another command.

14. Which command displays the first 10 lines of a file?

  • tail filename
  • head filename
  • more filename
  • less filename
Correct! The head command displays the first 10 lines of a file by default.

15. Which command displays the last 10 lines of a file?

  • tail filename
  • head filename
  • more filename
  • less filename
Correct! The tail command displays the last 10 lines of a file by default.
üîç Grep and Search

16. What does the grep command do?

  • Copies files
  • Moves files
  • Searches for patterns in text
  • Deletes files
Correct! The grep command searches for patterns in text files and displays matching lines.

17. What does the ^ symbol mean in grep?

  • End of line
  • Beginning of line
  • Any character
  • Negation
Correct! The ^ symbol matches the beginning of a line in grep patterns.

18. What does the $ symbol mean in grep?

  • End of line
  • Beginning of line
  • Any character
  • Negation
Correct! The $ symbol matches the end of a line in grep patterns.

19. Which command finds files and directories?

  • find
  • grep
  • cat
  • echo
Correct! The find command searches for files and directories based on various criteria.

20. What is the correct syntax for finding files by name?

  • find -name filename
  • find filename -name
  • find /path -name filename
  • find filename /path
Correct! The correct syntax is: find /path -name filename
‚úèÔ∏è Vi Editor

21. Which key enters insert mode in vi?

  • i
  • a
  • o
  • All of the above
Correct! All of these keys enter insert mode: 'i' (insert), 'a' (append), 'o' (open new line).

22. How do you save and quit vi?

  • :q
  • :wq
  • :w
  • :x
Correct! :wq saves the file and quits vi. :w saves, :q quits, :x also saves and quits.

23. Which key exits insert mode in vi?

  • Esc
  • Ctrl+C
  • Enter
  • Space
Correct! The Esc key exits insert mode and returns to command mode.

24. How do you search for text in vi?

  • :search text
  • ?text
  • /text
  • grep text
Correct! Use /text to search forward for text in vi. Use ?text to search backward.

25. How do you undo the last operation in vi?

  • :undo
  • u
  • Ctrl+Z
  • :u
Correct! Press 'u' in command mode to undo the last operation.

26. What is the first line of a bash script?

  • #!/bin/bash
  • #bash
  • #!/bash
  • /bin/bash
Correct! #!/bin/bash is the shebang line that tells the system which interpreter to use.

27. Which command makes a script executable?

  • chmod 755 script.sh
  • chmod +x script.sh
  • chmod u+x script.sh
  • All of the above
Correct! All of these commands make a script executable, with different permission levels.

28. How do you read user input in a bash script?

  • input variable
  • get variable
  • read variable
  • scan variable
Correct! Use 'read variable' to read user input into a variable.

29. How do you reference a variable in bash?

  • variable
  • $variable
  • &variable
  • @variable
Correct! Use $variable or ${variable} to reference a variable's value.

30. What does the 'alias' command do?

  • Creates shortcuts for commands
  • Copies files
  • Lists files
  • Deletes files
Correct! The alias command creates shortcuts or alternative names for commands.
